# 7-Day Launch Sprint

Goal: get a paid, China-friendly prototype online quickly without making unsafe claims.

## Day 1 - Prototype Baseline

- Finish local scanner.
- Finish one-click profile generation.
- Push private GitHub repo.
- Add CI.
- Prepare first desktop renderer.

## Day 2 - Desktop Shell

- Wire Electron scan, repair preview, and repair write actions.
- Add provider config screen.
- Store redacted local config.
- Add export report button.

## Day 3 - Website

- Landing page.
- Download page.
- Pricing page.
- Privacy policy.
- Terms.
- Contact channel.

## Day 4 - Packaging

- Add electron-builder or Tauri packaging.
- Build macOS DMG.
- Build Windows NSIS installer.
- Add app icon.
- Add unsigned internal download first.

## Day 5 - Payment

- Add Pro license code flow.
- Add Alipay/WeChat Pay provider decision.
- Add Stripe fallback for overseas users.
- Build manual activation code admin script if payment integration is not ready.

## Day 6 - China Region Prep

- Prepare ICP path if using mainland hosting.
- If no ICP yet, host outside mainland with China-accessible CDN.
- Add Chinese privacy policy and data retention statement.
- Prepare launch FAQ.

## Day 7 - Soft Launch

- Publish website.
- Upload macOS and Windows builds.
- Invite first 20 users.
- Collect scan failures and installation friction.
- Convert top support issues into automated checks.

## Launch Copy

Use:

- AI 工作环境体检
- 一键生成 AI 工作 Profile
- Windows / macOS 跨平台
- 本地隐私扫描
- 中英文 AI 工作流助手

Avoid:

- 防封号
- 绕过限制
- 反检测
- 伪装身份
- 保证账号安全
