# China Launch Checklist

## Positioning

Public name: VoiceShell AI Workspace / AI Workspace Doctor

Public promise:

> AI work environment health checks, one-click profiles, translation, and model configuration for Windows and macOS.

Do not use claims about bypassing, anti-ban, anti-detection, or guaranteed account safety.

## First Website Version

Pages:

- Home.
- Download.
- Pricing.
- Docs.
- Privacy policy.
- Terms.
- Contact.

Required copy:

- Local-first scanning.
- User-reviewed changes.
- No automatic system modification.
- No conversation upload by default.
- Support for user-owned API and network configuration.

## Distribution

### Windows

- Provide `.exe` installer from website first.
- Add Microsoft Store later if needed.
- Sign installer before paid launch.

### macOS

- Provide notarized `.dmg` from website first.
- Mac App Store China region can be evaluated later, but the app must avoid privileged system modification and unsupported claims.

## Payment

China-friendly options:

- WeChat Pay.
- Alipay.
- Stripe for overseas cards.
- Paddle or Lemon Squeezy if merchant onboarding is easier.

## Compliance Basics

- ICP filing if hosting a mainland China website.
- Public Security备案 if required by hosting setup.
- Chinese privacy policy.
- Clear data retention policy.
- Separate optional network checks from default local scan.

## Launch SKU

Free:

- Local scan.
- Basic report.

Pro:

Team:
